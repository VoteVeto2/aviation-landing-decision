# Long Landing Risk Attribution

A prototype research scheme for explaining and attributing long landing events from Quick Access Recorder (QAR) data in commercial aviation.

## Contents

- **long-landing-research-plan.html** — Complete high-level design, methodology, and reference implementation roadmap (English)
- **long-landing-research-plan-zh.html** — Chinese translation of the research plan

## Overview

This project presents a seven-stage attribution pipeline that:
1. Detects long-landing events from QAR data
2. Decomposes contributing factors using explainable ML (SHAP values)
3. Compares pilot technique against golden profiles
4. Generates actionable recommendations anchored to FAA/manufacturer guidance

Designed for US, EU, and Chinese aviation operators.
